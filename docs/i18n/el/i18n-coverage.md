# Χάρτης Κάλυψης i18n (Ελληνικά)

Αυτή η σελίδα είναι συνοπτική τοπικοποιημένη γέφυρα για τον πίνακα κάλυψης i18n.

Αγγλικό πρωτότυπο:

- [../../i18n-coverage.md](../../i18n-coverage.md)

## Σκοπός

- Να δείχνει ποιες γλώσσες χρησιμοποιούν bridge pages και ποιες έχουν βαθύτερη τοπικοποίηση
- Να καθοδηγεί ενημερώσεις coverage μετά από αλλαγές docs

## Σημείωση

- Η αγγλική τεκμηρίωση είναι το μόνο πλήρες runtime-contract tree.
- Οι μη αγγλικές γλώσσες, συμπεριλαμβανομένων των `vi` και `el`, χρησιμοποιούν hubs με bridge pages ή επιλεγμένους μεταφρασμένους οδηγούς ανά θέμα.
- Η κανονική κατάσταση κάλυψης παραμένει στο [../../i18n-coverage.md](../../i18n-coverage.md).
