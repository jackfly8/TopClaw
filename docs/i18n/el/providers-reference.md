# Αναφορά Providers TopClaw (Ελληνικά)

Αυτή είναι σελίδα-γέφυρα για IDs providers, aliases και απαιτήσεις authentication.

Αγγλικό πρωτότυπο:

- [../../providers-reference.md](../../providers-reference.md)

## Πότε να τη χρησιμοποιήσετε

- Για να επιλέξετε σωστό provider ID για `topclaw bootstrap` ή `topclaw agent`
- Για να ελέγξετε αν απαιτείται API key, OAuth ή subscription login
- Για να συγκρίνετε endpoint και model ιδιαιτερότητες

## Κανόνας

- Τα IDs provider, τα model names και οι κανόνες authentication ορίζονται στην αγγλική τεκμηρίωση.
- Αν υπάρχει διαφορά, ακολουθήστε το αγγλικό πρωτότυπο.
