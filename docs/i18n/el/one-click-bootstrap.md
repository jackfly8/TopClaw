# Εγκατάσταση με Μία Εντολή TopClaw (Ελληνικά)

Αυτή είναι σελίδα-γέφυρα για τη βασική ροή εγκατάστασης και bootstrap.

Αγγλικό πρωτότυπο:

- [../../one-click-bootstrap.md](../../one-click-bootstrap.md)

## Πότε να τη χρησιμοποιήσετε

- Για το ταχύτερο υποστηριζόμενο install path σε νέο σύστημα
- Για να αποφασίσετε ανάμεσα σε hosted installer και repo checkout
- Για να ελέγξετε τα flags των `bootstrap.sh` και `bootstrap.ps1`

## Κανόνας

- Η κανονική ροή εγκατάστασης ορίζεται στην αγγλική τεκμηρίωση.
- Η προτεινόμενη διαδρομή είναι ο επίσημος one-line installer και μετά το `topclaw bootstrap` για ρύθμιση.
