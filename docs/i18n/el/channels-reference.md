# Αναφορά Channels TopClaw (Ελληνικά)

Αυτή είναι σελίδα-γέφυρα για δυνατότητες channel, allowlists και βασικές οδηγίες ρύθμισης.

Αγγλικό πρωτότυπο:

- [../../channels-reference.md](../../channels-reference.md)

## Πότε να τη χρησιμοποιήσετε

- Για να επιλέξετε channel όπως Telegram, Discord, Slack ή WhatsApp
- Για να ελέγξετε ποια channels χρειάζονται background service
- Για να επιβεβαιώσετε routing ανά αποστολέα και group-reply συμπεριφορά

## Κανόνας

- Τα ονόματα channel, τα config keys και η ακριβής runtime συμπεριφορά ορίζονται στην αγγλική τεκμηρίωση.
- Για σφάλματα channel, συνδυάστε αυτή τη σελίδα με το [troubleshooting.md](troubleshooting.md) και το αγγλικό πρωτότυπο.
