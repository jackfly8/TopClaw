# Αναφορά Ρυθμίσεων TopClaw (Ελληνικά)

Αυτή είναι σελίδα-γέφυρα για βασικά config keys και defaults.

Αγγλικό πρωτότυπο:

- [../../config-reference.md](../../config-reference.md)

## Πότε να τη χρησιμοποιήσετε

- Για να επιβεβαιώσετε έγκυρα keys στο `config.toml`
- Για να ελέγξετε defaults ασφάλειας, service και channel
- Για να συγκρίνετε αλλαγές config μετά από upgrade

## Κανόνας

- Τα ονόματα των config keys παραμένουν στα αγγλικά.
- Η ακριβής runtime συμπεριφορά και οι λεπτομέρειες συμβατότητας ορίζονται στην αγγλική τεκμηρίωση.
