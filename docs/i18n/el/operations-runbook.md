# Runbook Λειτουργίας TopClaw (Ελληνικά)

Αυτή είναι σελίδα-γέφυρα για day-2 λειτουργία, background service και rollback.

Αγγλικό πρωτότυπο:

- [../../operations-runbook.md](../../operations-runbook.md)

## Πότε να τη χρησιμοποιήσετε

- Για εκκίνηση ή έλεγχο background service
- Για αποκατάσταση μετά από onboarding ή update
- Για αναζήτηση ασφαλών βημάτων rollback

## Κανόνας

- Οι τυπικές διαδικασίες λειτουργίας και ανάκτησης ορίζονται στην αγγλική τεκμηρίωση.
- Για επείγον χειρισμό, ακολουθήστε πρώτα το αγγλικό runbook.
