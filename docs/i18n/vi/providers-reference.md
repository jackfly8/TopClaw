# Tham khảo provider TopClaw (Tiếng Việt)

Đây là trang cầu nối cho danh sách provider, alias và ghi chú xác thực.

Nguồn tiếng Anh:

- [../../providers-reference.md](../../providers-reference.md)

## Dùng khi nào

- Chọn provider ID hợp lệ cho `topclaw bootstrap` hoặc `topclaw agent`
- Kiểm tra yêu cầu API key, OAuth hoặc subscription
- Đối chiếu endpoint/model đặc thù của từng provider

## Quy ước

- Provider ID, tên model, biến môi trường và hành vi xác thực chuẩn lấy theo bản tiếng Anh.
- Nếu có khác biệt giữa hai bản, ưu tiên tài liệu tiếng Anh.
