# Tham khảo cấu hình TopClaw (Tiếng Việt)

Đây là trang cầu nối cho các khóa cấu hình và giá trị mặc định quan trọng.

Nguồn tiếng Anh:

- [../../config-reference.md](../../config-reference.md)

## Dùng khi nào

- Kiểm tra khóa cấu hình hợp lệ trong `config.toml`
- Xác nhận giá trị mặc định về bảo mật, service và channel
- Đối chiếu thay đổi cấu hình sau khi nâng cấp TopClaw

## Quy ước

- Tên khóa cấu hình luôn giữ nguyên tiếng Anh.
- Hành vi runtime chính xác và mọi chi tiết tương thích ngược lấy theo bản tiếng Anh.
