# Mô hình runtime TopClaw (Tiếng Việt)

Đây là trang cầu nối giúp chọn giữa `agent`, `service`, `daemon` và `gateway`.

Nguồn tiếng Anh:

- [../../runtime-model.md](../../runtime-model.md)

## Dùng khi nào

- Chưa rõ nên chat trực tiếp trong terminal hay chạy service nền
- Muốn hiểu lúc nào `daemon` phù hợp hơn `service`
- Cần phân biệt đường đi giữa CLI, channel và gateway

## Quy ước

- Hành vi runtime chuẩn và các sơ đồ quyết định lấy theo tài liệu tiếng Anh.
- Với channel luôn bật như Telegram hoặc Discord, ưu tiên đọc kỹ phần `service` trong bản tiếng Anh.
